# 広告バナー設定方法

アプリの再ビルド・再配布は不要。MySQLに直接 INSERT するだけで反映されます。

## バナーを追加する

```sql
INSERT INTO banners (title, image_url, link_url, display_order)
VALUES ('タイトル', '画像URL', 'タップ時のリンクURL', 表示順);
```

- `title` — 管理用メモ（アプリには表示されない）
- `image_url` — バナー画像のURL
- `link_url` — タップ時の遷移先URL（空ならタップ無効）
- `display_order` — 表示順（小さい順）

## 期間限定バナー

```sql
INSERT INTO banners (title, image_url, link_url, display_order, start_date, end_date)
VALUES ('4月限定', 'https://example.com/april.png', '', 1, '2026-04-01 00:00:00', '2026-04-30 23:59:59');
```

## バナーを非表示 / 再表示 / 削除する

```sql
UPDATE banners SET is_active = 0 WHERE id = 1;  -- 非表示
UPDATE banners SET is_active = 1 WHERE id = 1;  -- 再表示
DELETE FROM banners WHERE id = 1;                -- 削除
```

## 現在のバナー一覧を確認する

```sql
SELECT * FROM banners ORDER BY display_order;
```

## 補足

- 複数登録すると自動スライド（5秒間隔）のカルーセル表示になる
- 推奨画像サイズ: 横1200px以上、縦400px程度（アスペクト比 約3:1）
- 画像読み込み失敗時は「キャンペーン・広告エリア」と表示される
